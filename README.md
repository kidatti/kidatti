## 🚀 主なプロジェクト

### Network / Ops
- **netchecker**  
  CLIとWeb UIの両方を備えたネットワーク診断ツール

- **localscan**  
  高速なローカルネットワークスキャンツール（複数方式・複数フォーマット対応）

- **httplog**  
  HTTPリクエスト（ヘッダ・クエリ・ボディ）を記録できるログサーバ

- **curl-batch**  
  CSVとテンプレートからHTTPリクエストを一括実行するツール

### Chrome Extensions
- **chrome-web-local-tools**  
  JSON / JWT / ハッシュ / エンコードなどをまとめた開発者向けツール集

- **chrome-copy-tabs**  
  タブのコピー・マーキング・エクスポートを行う拡張

- **chrome_aws_switch_role_links**  
  `.aws/config` ベースで AWS ロール切り替えを補助する拡張

### Web / Cloud
- **proxy-cloudflare-workers**  
  APIキー対応のシンプルなプロキシAPI（Cloudflare Workers）

- **godbadmin**  
  MySQL / PostgreSQL / MariaDB 対応のWeb DB管理ツール
